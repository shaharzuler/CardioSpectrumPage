Description

The ability to map left ventricle (LV) myocardial motion using computed tomography angiography (CTA) is essential to diagnosing cardiovascular conditions and guiding interventional procedures. Our project presents a comprehensive approach to address this problem using 3D optical flow techniques and Functional Maps (FMs).

Installation

git clone https://github.com/shaharzuler/cardio_volume_skewer.git
cd cardio_volume_skewer
pip install -r requirements.txt
